## How to Run Backend
cd backend
npm install
npm start

Runs on:
[http://localhost:5000](http://localhost:5000)

> Note: Ensure your `.env` file contains your MongoDB connection string:
> ```
> MONGO_URI=<your_mongo_uri_here>
> ```

## How to Run Frontend
cd frontend
npm install
npm start

Runs on:
[http://localhost:3000](http://localhost:3000)
